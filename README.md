<h2 align="center">転職サプリ</h2>

## 🌐 App URL

### **https://job-switch.herokuapp.com/**  
　
## 💬 Usage

`$ git clone https://github.com/saganmon/switching-jobs.git`
<br>
`$ cd switching-jobs`
<br>
`$ bundle install`
<br>
`$ rails db:create`
<br>
`$ rails db:migrate`
<br>
`$ rails s`
<br>
👉 http://localhost:3000
